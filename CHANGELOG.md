# CHANGELOG


## v2.0.0 (2026-03-30)

### Features

- Complete rewrite using Streamlit Custom Components v2 (CCv2) — no more iframes
- Modern Vite + ESM build replaces legacy webpack/CRA toolchain
- CSS `--st-*` variable theming replaces manual ThemeParser
- Collector-based architecture for flexible data return processing
- Renamed project from `streamlit-aggrid` to `st-aggrid`
